# Netflix Data Analysis Dashboard

## Project Overview
This project analyzes the Netflix dataset to uncover valuable insights into content trends, ratings, genres, countries, release years, and directors. The dataset was cleaned, transformed, and visualized to support data-driven decision-making.

## Tools Used
- Microsoft Excel
- SQL (MySQL)
- Python (Pandas, NumPy, Matplotlib)
- Power BI

## Key Features
- Data Cleaning and Preprocessing
- SQL Queries for Business Insights
- Interactive Power BI Dashboard
- KPI Cards and Charts
- Content Distribution Analysis
- Movies vs TV Shows Comparison
- Country-wise Content Analysis
- Release Year Trends
- Rating Distribution

## Business Insights
- Identified the distribution of Movies and TV Shows.
- Analyzed top content-producing countries.
- Studied content trends over the years.
- Explored rating categories and genre distribution.
